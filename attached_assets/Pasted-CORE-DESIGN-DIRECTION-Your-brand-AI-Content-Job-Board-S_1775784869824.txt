🧩 SHADCN COMPONENTS YOU SHOULD USE

Install:

npx shadcn-ui@latest add button card input textarea dialog tabs badge skeleton
💎 PREMIUM TOUCHES (THIS IS WHAT MOST PEOPLE MISS)
1. Micro Animations
Buttons → slight scale on hover
Cards → subtle lift
hover:scale-[1.02] transition
2. Gradient Glow (very subtle)

For primary elements:

bg-gradient-to-r from-purple-600 to-indigo-600
3. Skeleton Loading

Instead of spinners:

Use grey blocks (feels pro)
4. Empty States

Don’t leave blank pages.

Example:

“No tasks yet. Be the first to post one.”

🚫 WHAT TO AVOID
Too many colors
Big shadows
Cluttered UI
Tiny fonts
Centered everything
